FCO Ambassador Twitter Networks

Preliminary social network analysis using Twitter data. Follower information is gathered and processed using a python script and visualised with the open source tool Gephi. Rudimental community detection algorithms are implemented and the layout of the networks are dictated by Force Atlas and Fruchterman-Reingold models. Specific details will be given in the About section which is linked to at the top of the page.
